<h1>Guía de Inicio para Desarrolladores</h1>

<p>
  El Sistema de Diseño Phoenix contiene una colección de componentes de React
  que se pueden utilizar para construir aplicaciones de React accesibles y de
  alto rendimiento, consistentes con las pautas de marca de Phoenix. Esta guía
  de inicio proporciona las instrucciones básicas para instalar los componentes
  del Sistema de Diseño Phoenix como una dependencia en tu aplicación y comenzar
  a usar esos componentes. La documentación de cada componente individual
  proporciona información más detallada, que incluye:
</p>
<ul>
  <li>
    Una descripción general con orientación de uso contextual y mejores
    prácticas.
  </li>
  <li>
    Ejemplos de variaciones comunes del componente, como tamaños y estados.
  </li>
  <li>
    Orientación para utilizar los componentes teniendo en cuenta la
    accesibilidad.
  </li>
  <li>
    Una descripción detallada de las props de React disponibles, junto con un
    entorno de juego interactivo.
  </li>
</ul>
