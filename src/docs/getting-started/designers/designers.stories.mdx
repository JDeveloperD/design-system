<h1>Guía de Inicio para Diseñadores</h1>

<p>
  El Sistema de Diseño Phoenix se utiliza para comunicar la identidad de marca y
  los componentes de interfaz de usuario de Phoenix. Las interfaces de usuario
  de Phoenix son tan diversas como sus usos, que van desde sitios web
  inspiradores hasta aplicaciones para un servicio específico. El objetivo es
  crear soluciones que proporcionen una experiencia de usuario equilibrada en
  todo el sistema, que se transicione sin problemas desde la aplicación hasta el
  vehículo utilizando un conjunto central de componentes, módulos y animaciones.
  El Kit de Diseño Phoenix actúa como la base, que contiene los componentes
  principales y los tokens que se pueden aplicar a todos los diseños de interfaz
  de usuario creados para la suite de productos digitales de Phoenix. La Guía de
  Inicio que se describe en esta página proporciona instrucciones sobre cómo
  instalar el Kit de Diseño Phoenix e implementarlo en tu proyecto.
</p>
